# AI Solution Developer - Samples & Labs

Acest repository conține laboratoare și exemple de cod pentru cursul *AI Solution Developer*.

## Setup

1. Instalați [Git](https://git-scm.com/) 
2. Clonați acest repository:

    ```
    git clone https://github.com/neaorin/ai-developer-samples
    ```

3. Navigați în interiorul directorului clonat:

    ```
    cd ai-developer-samples
    ```

4. (opțional) Deschideți IDE-ul preferat în acest director. Dacă folosiți [Visual Studio Code](https://code.visualstudio.com/), puteți face asta folosind comanda

    ```
    code .
    ```
